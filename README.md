# LogServer
- Elixir Application module used to start a GenServer based UDP Server on a user defined port
- can pass log handler callback as `:log_handler` application environment config

### Example config
```
	config :log_server,
	ip: {127,0,0,1},
	port: 1514,
	log_handler: {module, function}
```
